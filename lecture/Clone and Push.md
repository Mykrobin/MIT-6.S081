# 连接xv6 与 个人仓库

## 精简操作

查看远程仓库 `git remote -v`

添加MIT远程仓库 ` git remote add labs git://g.csail.mit.edu/xv6-labs-2020`

拉取分支 ` git fetch labs`

查看分支 `git branch -a`

切换分支 `git checkout util`

提交代码到个人仓库 `git push https://github.com/Mykrobin/MIT-6.S081.git` 或者 `git push git@github.com:Mykrobin/MIT-6.S081.git`
